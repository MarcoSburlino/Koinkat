# Screenshot assets for the README guide

Images referenced by the installation and setup guide in the repository
README. The README contains HTML comment placeholders pointing at these
exact filenames; drop the image in with the matching name and add the
Markdown image line in place of the comment - no restructuring needed.

Naming convention: `NN-short-kebab-name.png`, numbered in the order the
guide reaches them.

| File | What to capture |
|---|---|
| `01-first-launch.png` | First launch: the user-profile (name) step |
| `02-workspace-hub.png` | Workspace hub with the creation cards |
| `03-bank-credentials.png` | Bank-linked workspace wizard: credential fields |
| `04-bank-setup-guide.png` | The in-app Enable Banking setup guide |
| `05-consent-flow.png` | Bank consent page or the callback page with the Open Koinkat prompt |
| `06-dashboard.png` | Dashboard with accounts and the month pulse card |

Capture from a demo or sandbox workspace only - fixture data, nothing
personal. PNG preferred; keep widths around 1200-1600 px so text stays
readable on the GitHub-rendered page.
